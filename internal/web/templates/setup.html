{{end}}
  • One-click deploy: The catalog deploys apps directly to your OpenHost instance. Requires permission to read the router token from the Secrets app.
  • Manual install links: The catalog links you to the OpenHost dashboard where you install apps yourself. No router token needed.

Impact of changing mode

This only affects how the catalog deploys new apps. It does not change your OpenHost login, password, or any already-deployed apps. The router token is never stored by the catalog — it is re-resolved from the Secrets app on each deploy.
